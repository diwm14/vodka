- Member, Group, Boardgame ...

- 하상경, 그룹3, 테포마 ...

### action 명세
- 나누어진 데이터를 리스트로 보여주기
- 리스트에 추가하거나 수정하기.
  - 리스트에 추가하거나 수정하게 되면 state에 변경 필요.
